# 📈 StockVision Pro: Market Intelligence Terminal

**StockVision Pro** is a sophisticated financial data analysis platform built with Python and Streamlit. It provides a professional-grade "Dark Mode" terminal experience for real-time market tracking and advanced stock performance visualization.

## ✨ Key Features

*   **Live Market Snapshot:** A real-time header displaying the latest prices and percentage changes for major tickers (AAPL, MSFT, NVDA, etc.).
*   **Advanced Charting:** Support for three distinct visualization types:
    *   **Candlestick Charts:** For price action and volume analysis.
    *   **Line Charts:** With integrated 7-day and 20-day Moving Averages (MA).
    *   **Area Charts:** For clean, high-level trend visualization.
*   **Deep Analytics & Statistics:** Automated calculation of Daily Returns distribution, Volatility (Daily Std Dev), and Period High/Low.
*   **Interactive Historical Data:** A full OHLCV (Open, High, Low, Close, Volume) data table with sorting and filtering capabilities.
*   **High-End UI/UX:** Custom-themed interface using Google Fonts (Space Mono & Syne) for a futuristic, technical aesthetic.

## 🛠️ Tech Stack

*   **Streamlit:** For the interactive web framework and dashboard layout.
*   **yFinance:** For fetching reliable, real-time market data from Yahoo Finance.
*   **Plotly:** For high-performance, interactive financial visualizations.
*   **Pandas:** For complex financial data manipulation and analysis.
